# 是什么

JDBC

JDBC 是 Java DataBase Connection 的缩写, 顾名思义, JDBC赋予了Java和数据库进行通信的能力. 学了JDBC, 我们就可以通过Java代码在后端对数据库进行各种操作.

之前我们通过HTML等知识写一些静态网页, 这些网页的数据是固定, 而通过和数据库进行连接, 便可以让网页上的数据"动起来".

可以说, 学习JDBC是学习JavaWeb的第一步!

MyBatis

MyBatis是对JDBC的封装. 在JDBC中, 总有一些通用的操作, 比如, 连接数据库, 关闭资源, 增删改查操作等等. JDBC对此进行了封装, 可以让我们更加方便地使用JDBC, 避免每个项目都要写一些重复的代码.
