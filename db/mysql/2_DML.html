# 数据操纵

数据操纵语言用于对数据库中的数据进行增删改操作, 有:

  • 插入: INSERT

  • 修改: UPDATE

  • 删除: DELETE

# 增加数据

# 第一种插入格式

-- 案例: 给student表插入一个新的学生
INSERT INTO student
VALUES (4, '李白', 18, '男');

上述例子中, 我们按顺序给每一列赋值, 但是, 有时候有的列的值可能暂时为空, 此时我们可以先将其赋值为NULL

-- 案例: 7号学生选修了3号课程, 请插入这组数据(由于还没考试, 所以不知道分数)
INSERT INTO score
VALUES (7, 3, null);

我们也可以自行选择对哪些项赋值, 以及先对哪个项赋值, 剩余的被自动赋值为默认值(默认值是自己建表时设置的)或NULL . 要注意这样的话列的数量和值的数量必须相同.

-- 案例: 插入一个学生, 但是我们还不知道他的姓名和性别
INSERT INTO student(s_id, s_age)
VALUES (10, 18);
-- 或
INSERT INTO student(s_age, s_id)
VALUES (18, 11);

我们这里并没有讨论数据类型的问题, 但是要注意在建表的时候每一列都会被设置好相应的数据类型, 我们在赋值时要遵循这些数据类型(比如字符串要用单引号或双引号括起来, 日期的格式应该如'1999-12-2'这样). 在DDL中会有更详细的介绍.

另外有些列的值可以是空值NULL , 而有的则不行, 这些也会在DDL中介绍.

# 修改数据

修改用UPDATE 语句实现, 我们可以结合WHERE 语句来筛选要修改的列

-- 修改成绩表所有人的成绩为0
UPDATE score
SET s_score = 0;

-- 修改姓为'张'的女生的年龄为18岁
UPDATE student
SET s_age = 18
WHERE s_name LIKE '张%';

也可以用连接语句对某表进行修改

-- 修改1号老师的课程的成绩为0
UPDATE course
LEFT JOIN score
ON course.c_id = score.c_id
SET score.s_score = 0
WHERE course.t_id = 1;
