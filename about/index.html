# 不同的状态

有如下不同的状态，分别表示不同的含义

  1. 暂无——想学习这方面的知识，不过目前暂时没空，以后可能更新，也可能不更新
  2. 即将上线——正在学习或即将学习这方面的知识，不过还没有上线相关文档
  3. 未完待续——相关文档已经上线，不过整体来说不够完善或结构不够清晰。也就是说，尽管能看，但是不推荐阅读
  4. GA——稳定版本，基本整理好了，不过以后可能会进行小的调整，或往里面增加新的主题，或移动文档的结构
  5. 停更——由于某种原因，不再关注相关内容，所以内容可能会过时，不过日后不排除重新更新的可能
